# 🏠 Hyderabad House Price Prediction

## 📌 Project Overview
This project predicts house prices in Hyderabad using Machine Learning and Streamlit.

## 🚀 Features
- Predict house prices based on:
  - Location
  - Area (sq ft)
  - BHK
  - Bathrooms
- Interactive web interface using Streamlit

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## 📂 Project Structure
House_Price_Prediction/
│── GUI/
│── data/
│── models/
│── training.ipynb
│── requirements.txt

## ▶️ Run the Project

```bash
pip install -r requirements.txt
streamlit run GUI/app.py
```
